Maintenance Help

This section provides tools to manage your wine inventory database and its synchronization with the Home Assistant ToDo list.

  • Sync ToDo from DB: This button will synchronize your entire wine inventory from the Wonderful-Wino database to your ToDo list. It attempts to update or re-add every wine from your inventory into the list.

    Important Note: If your ToDo list contains items that are no longer in your inventory, the "Sync" operation will not remove them. You can clear these manually by “completing” them via the ToDo list in the same manner as when consuming a wine.

  • Reset DB: WARNING: This action will permanently delete ALL your wine data from the Wonderful-Wino database. This cannot be undone. Use this only if you want to start your wine inventory from scratch.
