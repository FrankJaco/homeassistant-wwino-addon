Maintenance Help

This section provides tools to manage your wine inventory database and its synchronization with the Home Assistant ToDo list.

  • Backup Database: Creates a safe copy of your entire wine inventory. The backup is stored as `wonderful_wino_backup.db` inside your Home Assistant `/share/wwino/` folder. You can access this file using the Samba or File Editor add-ons. This is useful for creating a restore point before making major changes.
  • Restore Database: WARNING: This will completely overwrite your current inventory with the data from the `wonderful_wino_backup.db` file. Any wines added since your last backup will be lost. Use this to recover your data if something goes wrong.
  • Sync DB→ToDo: This button will synchronize your entire wine inventory from the Wonderful-Wino database to your ToDo list. It attempts to update or re-add every wine from your inventory into the list.

    Note: This operation will not remove items from the ToDo list that are no longer in your inventory.

  • Reset DB: WARNING: This action will permanently delete ALL your wine data from the Wonderful-Wino database. This cannot be undone. Use this only if you want to start your wine inventory from scratch.
